Veo API Research Findings for Muse Video Generation Upgrade
=============================================================

Model: Veo 3.1 (veo-3.1-generate-preview)
- Generates 8-second 720p/1080p/4K videos with natively generated audio
- Supports portrait (9:16) and landscape (16:9) formats
- Supports text-to-video, image-to-video, and video extension
- Can use up to 3 reference images to guide content
- Supports dialogue generation with audio

JavaScript SDK: @google/genai
Usage:
  import { GoogleGenAI } from "@google/genai";
  const ai = new GoogleGenAI({});
  let operation = await ai.models.generateVideos({
    model: "veo-3.1-generate-preview",
    prompt: prompt,
  });
  // Poll until done
  while (!operation.done) {
    await new Promise(resolve => setTimeout(resolve, 10000));
    operation = await ai.operations.getVideosOperation({ operation });
  }
  // Download
  ai.files.download({
    file: operation.response.generatedVideos[0].video,
    downloadPath: "output.mp4",
  });

Config options:
- personGeneration: "allow_all" | "allow_adult" | "dont_allow"
- aspectRatio: "16:9" | "9:16"
- numberOfVideos: 1-2
- resolution: "480p" | "720p" | "1080p" | "4k"
- duration: 5 or 8 seconds
- enhancePrompt: true/false

For Muse integration plan:
